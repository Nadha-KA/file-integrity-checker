# File Integrity Checker

## Overview
This project is a Python-based file integrity checker that uses SHA-256 hashing to detect file modifications or tampering.

The program generates a digital fingerprint (hash) of a file and compares it later to verify whether the file has been altered.

## Features
- Calculates SHA-256 hash of files  
- Detects file modification or tampering  
- Uses secure binary file reading  
- Simple and beginner-friendly cybersecurity implementation  

## Technologies Used
- Python  
- hashlib (Hashing Library)  

## Purpose
This project was developed as a beginner-level cybersecurity exercise to understand data integrity verification using cryptographic hashing.

## How to Use
1. Create a text file (example: test.txt)  
2. Write some content inside the file  
3. Run the Python program  
4. Enter the file name when prompted  
5. Modify the file and test integrity detection
